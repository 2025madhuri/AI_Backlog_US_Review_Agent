# AI Delivery Intelligence Agent

## Overview

AI Delivery Intelligence Agent is an autonomous sprint analytics and reporting system that integrates with Jira, analyzes sprint disruptions, generates AI-based retro summaries, creates PDF reports, and automatically emails delivery intelligence reports.

---

## Features

- Jira Integration
- Sprint Capacity Analysis
- Unplanned Work Detection
- QA Leakage Detection
- Hotfix Tracking
- AI Retro Summary Generation
- PDF Report Generation
- Automated Email Notification
- Streamlit Dashboard

---

## Technologies Used

- Python
- Streamlit
- Jira Python SDK
- ReportLab
- SMTP / Gmail
- VS Code
